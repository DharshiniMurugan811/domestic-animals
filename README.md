The farm home is a responsive website that provides information about common domestic animals and their importance in human life. The website also allows users to place orders for animals and farm products through Whatsapp.

Features are, 
  Resposive design for desktop, tablet, laptop and mobile devices, 
  Attractive animal cards with images,
  Whatsapp order button for easy communication, and
  Modern layout using HTML5 and CSS3 with media queries.
